# shoot-contributions
Turn your GitHub contributions into a playable space shooter game, updated automatically by GitHub Actions.

## ■ My GitHub Activity Game

<p align="center">
  <img src="assets/space-shooter.gif" />
</p>

## ■ How It Works

- Reads your public GitHub contributions.
- Converts your contribution graph into a space shooter level.
- Renders an animated GIF and saves it to `assets/space-shooter.gif`.
- Updates daily at midnight (UTC) or whenever you run the workflow manually.

## ■ Tech Behind This Repo

- GitHub Actions
- [`czl9707/gh-space-shooter`](https://github.com/czl9707/gh-space-shooter) GitHub Action
- Auto-generated assets stored in this repository
